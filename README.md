# OIBSP_DataAnalytics_Task3_Wine_Quality_Prediction# Wine Quality Prediction (Machine Learning Project)

Objective
To build a Machine Learning model that predicts wine quality based on physicochemical properties.

 Tools & Technologies Used
 Python
 Pandas
 NumPy
 Matplotlib & Seaborn
 Scikit-learn
 Jupyter Notebook
 Streamlit (for deployment)

 Steps Performed
 Data Cleaning
 Exploratory Data Analysis (EDA)
 Feature Selection
 Model Building using Machine Learning algorithms
 Model Evaluation
 Model Deployment using Streamlit

ML Libraries Used
sklearn.model_selection
sklearn.linear_model / RandomForestClassifier
sklearn.metrics
pickle (for model saving)

## Outcome
Successfully built and deployed a predictive model that estimates wine quality with good accuracy.
